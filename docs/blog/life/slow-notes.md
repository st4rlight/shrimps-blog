# 慢一点，也是在前进

最近越来越觉得，很多事情不一定要追求立刻见效。

学习、写作、做项目，很多时候都更像是长期积累。

短时间内看不到明显变化，不代表没有成长。

## 我想提醒自己的几件事

- 不要因为慢就否定自己
- 保持节奏比偶尔爆发更重要
- 记录下来，能帮助自己看见变化

这座博客大概也会成为这样的地方。

它不追求一次性完成，而是允许我慢慢把内容写长、写深、写成体系。
