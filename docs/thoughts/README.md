# 思考领悟

这里记录我近一段时间的总结反思，以及对一些事情的看法、理解和想法。

不是每一次思考都需要写成完整的技术文章，很多时候，一段随想、一个感悟、一次复盘，都值得被留下来。

## 写什么

- 阶段性的总结与反思
- 对一些事情的看法和理解
- 生活中的灵感与感悟
- 读书、观影后的思考

## 为什么写

写下来，是为了让自己看见变化。

很多想法如果不及时记录，就会在忙碌中被遗忘。这里不是追求完美的输出，而是给自己一个持续思考的空间。
