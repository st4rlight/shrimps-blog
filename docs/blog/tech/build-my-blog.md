# 【AI生成测试文章】我为什么开始搭建这座博客

很多内容如果不写下来，就会很快消失。

我想要一个属于自己的地方，能长期积累下面这些东西：

- 学到的知识
- 做项目时踩过的坑
- 一些突然冒出来但值得继续展开的想法

相比把内容完全散落在不同平台，我更希望有一个可以自己掌控结构、风格和发布节奏的站点。

这也是我选择自己搭博客的原因。

## 这座博客会写什么

- 技术文章
- 项目复盘
- 工具使用经验
- 生活里的阶段性记录

## 我对它的期待

它不一定每天更新，但希望一直存在。

只要持续写下去，这里就会慢慢长成一份属于 `st4rlight` 的数字轨迹。
